Design Pad Help

Hello, and welcome to designPad a 21st century Ui and diagram designer. Here you can design all sorts of diagrams and User Interface for your application.

  • Drag and drop the shapes and diagrams form left panel to the workspace.
  • To import images, just drap and drop it from pc to the workspace.
  • Export the designs as png, SVG and json forma.t
  • Save it in json format to continue it later

  • Ctrl + C : Copy
  • Ctrl + V : Paste
  • Ctrl + X : Cut
  • Ctrl + G : Group
  • Ctrl + U : UnGroup
  • Ctrl + F : Bring To Front
  • Ctrl + B : Send To Back
